# OpenAPI Inputs

This directory stores the checked-in admin control-plane OpenAPI snapshots used by `sdkwork-im-admin-sdk`.

- authority source: `http://localhost:3000/admin/im/v3/openapi.json`
- local copies:
  - `openchat-im-admin.openapi.yaml`
  - `openchat-im-admin.sdkgen.yaml`

The runtime schema is the source of truth. Checked-in files are refreshed by the generation wrappers.

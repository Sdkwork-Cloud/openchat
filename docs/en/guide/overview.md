# 项目概览

OpenChat 是一个**开源即时通讯解决方案**，提供从服务端到客户端的完整技术栈，帮助开发者快速构建企业级即时通讯应用。

## 什么是 OpenChat？

OpenChat 不仅仅是一个聊天服务端，它是一个**完整的即时通讯生态系统**：

- **服务端** - 基于 NestJS 的高性能后端服务
- **SDK** - 多语言客户端 SDK（TypeScript、Java、Go、Python）
- **应用** - 开箱即用的示例应用（Web、PC、移动端）
- **集成** - 与悟空IM深度集成，支持第三方平台

## 核心特性

### 🚀 即时通讯

- **单聊** - 一对一私聊，支持文字、图片、语音、视频、文件
- **群聊** - 多人群组聊天，支持群管理、禁言、@功能
- **消息特性** - 已读回执、消息撤回、消息搜索、历史消息
- **离线消息** - 支持离线消息推送和同步

### 🔊 实时音视频

- **音视频通话** - 基于火山引擎/腾讯云 RTC
- **群组通话** - 支持多人音视频通话
- **屏幕共享** - 支持屏幕共享和远程协作
- **互动直播** - 支持低延迟直播和互动

### 🤖 AI 能力

- **AI Bot** - 内置 GPT 等大模型支持
- **智能客服** - 自动回复常见问题
- **智能问答** - 基于知识库的智能问答

### 🔌 第三方集成

- **Telegram** - 同步 Telegram 消息
- **WhatsApp** - 同步 WhatsApp 消息
- **Webhook** - 支持自定义 Webhook 集成

## 快速体验

### 在线 Demo

- **Web 端**: [https://demo.openchat.dev](https://demo.openchat.dev)
- **管理后台**: [https://admin.openchat.dev](https://admin.openchat.dev)

### 本地部署

```bash
# 克隆项目
git clone https://github.com/openchat-team/openchat-server.git
cd openchat-server

# 一键部署
chmod +x scripts/quick-start.sh
./scripts/quick-start.sh
```

部署完成后访问：
- OpenChat API: http://localhost:3000
- 悟空IM Demo: http://localhost:5172
- 管理后台: http://localhost:5300/web

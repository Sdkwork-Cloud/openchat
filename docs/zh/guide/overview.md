# 项目概览

OpenChat 是一个**开源即时通讯解决方案**，提供从服务端到客户端的完整技术栈，帮助开发者快速构建企业级即时通讯应用。

## 什么是 OpenChat？

OpenChat 不仅仅是一个聊天服务端，它是一个**完整的即时通讯生态系统**：

- **服务端** - 基于 NestJS 的高性能后端服务
- **SDK** - 多语言客户端 SDK（TypeScript、Java、Go、Python）
- **应用** - 开箱即用的示例应用（Web、PC、移动端）
- **集成** - 与悟空IM深度集成，支持第三方平台

## 快速体验

### 本地部署

```bash
# 使用安装向导
curl -fsSL https://raw.githubusercontent.com/Sdkwork-Cloud/openchat/main/scripts/setup-wizard.sh | bash

# 或 Docker Compose
git clone https://github.com/Sdkwork-Cloud/openchat.git
cd openchat
docker compose up -d
```

部署完成后访问：
- OpenChat API: http://localhost:3000
- API 文档: http://localhost:3000/api/docs
- WukongIM Demo: http://localhost:5172

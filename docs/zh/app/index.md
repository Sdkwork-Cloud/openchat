# 应用概览

OpenChat 提供多平台应用示例，帮助开发者快速上手。

## 支持的平台

| 平台 | 技术栈 | 状态 |
|------|--------|------|
| **Web** | React + TypeScript | ✅ 已支持 |
| **PC** | React + Electron | ✅ 已支持 |
| **iOS** | React Native | ✅ 已支持 |
| **Android** | React Native | ✅ 已支持 |
| **小程序** | 微信小程序 | ✅ 已支持 |

## 应用列表

- [React Native](./react-native) - 移动端应用
